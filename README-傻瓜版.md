# A股智能助手 - 傻瓜版介绍

## 1. 项目简介

一款帮你看股票、分析行情、用AI给投资建议的免费桌面软件。

## 2. 主要功能

- 📊 看大盘：上证指数、深证成指、创业板指实时数据
- 📈 看个股：自选股管理、K线图、技术指标
- 🤖 AI分析：智能选股、风险评估、买卖建议
- 📚 知识库：投资知识学习、文档管理
- ⚙️ 简单配置：一键安装、自动启动

## 3. 快速开始（3步走）

**第1步：安装**  
双击 `install.bat`，等2-5分钟装好。

**第2步：启动**  
双击 `一键启动-标准模式.bat` 打开软件。

**第3步：使用**  
- 点击"行情看板"搜股票加到自选
- 点击"AI助手"问问题、看分析
- 点击"仪表盘"看大盘整体情况

## 4. 适用人群

- 股票新手：想看懂行情又不会分析
- 上班族：没时间天天盯盘
- 技术小白：不会用复杂的股票软件
- 想学习：边用边学投资知识

## 5. 一句话总结

普通人也能用的免费AI股票助手，3分钟上手，让投资更简单。
