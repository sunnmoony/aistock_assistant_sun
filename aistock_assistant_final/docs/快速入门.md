## 安装步骤

### 方式一：一键安装（推荐新手）

1. **双击运行 `install.bat`**
   - 程序会自动检测Python环境
   - 自动安装所需依赖
   - 创建必要的配置文件

2. **等待安装完成**
   - 安装过程约需2-5分钟
   - 看到提示"安装完成"即可

### 方式二：手动安装

```bash
# 1. 安装依赖
pip install -r requirements_minimal.txt

# 2. 复制配置文件
copy config.example.yaml config.yaml
```

### 系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Windows 7 | Windows 10/11 |
| Python | 3.8 | 3.10+ |
| 内存 | 2GB | 4GB+ |
| 硬盘 | 500MB | 1GB |

## 基本操作

### 1. 添加自选股

**方法一：搜索添加**
1. 点击左侧"行情看板"
2. 在搜索框输入股票代码或名称
3. 点击搜索结果中的"添加"按钮

**方法二：直接输入**
1. 在行情页面找到"添加股票"按钮
2. 输入6位股票代码（如：600000）
3. 点击确认

### 2. 查看股票分析

1. 在自选股列表中点击股票
2. 右侧信息面板显示详细信息
3. 点击"AI分析"按钮获取智能分析

### 3. 刷新数据

- **手动刷新**：点击工具栏"刷新"按钮
- **自动刷新**：程序每30秒自动刷新一次

## 配置AI功能（可选）

### 步骤

1. 访问 [硅基流动](https://siliconflow.cn/)
2. 注册账号并获取API密钥
3. 打开 `config.yaml` 文件
4. 填写API密钥：

```yaml
ai:
  api_key: "your_api_key_here"
```

## 常见问题

### Q: 程序启动失败？

**解决方案**：
1. 确认已运行 `install.bat`
2. 检查Python版本是否为3.8+
3. 尝试使用 `run_lite.bat` 轻量模式

### Q: 无法获取股票数据？

**解决方案**：
1. 检查网络连接
2. 程序会自动切换数据源
3. 运行 `data_source_wizard.py` 重新配置数据源

### Q: AI功能不可用？

**解决方案**：
1. 确认已在 `config.yaml` 中配置API密钥
2. 检查API密钥是否有效
3. 确认网络可以访问API服务

### Q: 程序运行卡顿？

**解决方案**：
1. 使用 `run_lite.bat` 轻量模式
2. 减少自选股数量
3. 关闭其他占用内存的程序
