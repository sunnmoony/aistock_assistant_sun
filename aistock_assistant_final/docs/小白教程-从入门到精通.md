## 一、准备工作

### 1.1 系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Windows 7 | Windows 10/11 |
| Python | 3.8 | 3.10+ |
| 内存 | 2GB | 4GB+ |
| 硬盘 | 500MB | 1GB |

### 1.2 安装步骤（傻瓜式一键安装）

**超级简单，只需两步！**

#### 第一步：双击安装

找到项目文件夹中的 **`install.bat`** 文件，双击运行！

- 程序会自动检测您的电脑环境
- 自动下载和安装所需的软件包
- 自动创建必要的配置文件

**等待2-5分钟，看到"安装完成"提示就OK了！**

#### 第二步：启动程序

安装完成后，选择以下任一方式启动：

- **标准模式**：双击 `一键启动-标准模式.bat`（推荐）
- **轻量模式**：双击 `一键启动-轻量模式.bat`（电脑配置较低时使用）

## 五、如何添加自选股

### 方法一：通过搜索框添加（推荐）

**步骤：**

1. 点击左侧导航栏的 **"📈 行情看板"**
2. 在顶部搜索框中输入：
   - 股票代码（如：600519）
   - 或股票名称（如：贵州茅台）
3. 按 **回车键** 或点击搜索
4. 点击 **"➕ 添加自选"** 按钮
5. 看到"已添加到自选股"提示就成功了！

### 方法二：通过快捷按钮添加

**步骤：**

1. 在 **"📊 仪表盘"** 页面
2. 点击 **"➕ 添加自选"** 快捷按钮
3. 在弹出的对话框中输入股票代码
4. 点击确认

### 如何删除自选股

**步骤：**

1. 进入 **"📈 行情看板"** 页面
2. 在自选股表格中找到要删除的股票
3. 点击该行右侧的 **"🗑️ 删除"** 按钮
4. 在确认对话框中点击 **"是"**

### 小技巧

- **添加多只股票**：重复上述步骤，可以添加多只股票到自选股
- **查看自选股**：在筛选下拉框中选择"自选股"，只显示您关注的股票
- **建议数量**：自选股建议不超过20只，方便管理和关注

## 六、如何使用AI分析

### 6.1 前置准备：配置API密钥

**重要提示：使用AI功能前需要先配置API密钥！**

#### 获取API密钥

1. 访问 [硅基流动](https://siliconflow.cn/) 网站
2. 注册一个账号（免费）
3. 登录后进入控制台
4. 找到"API密钥"管理页面
5. 点击"创建API密钥"
6. 复制生成的API密钥（格式类似：sk-xxxxxxxxxx）

#### 配置API密钥

**方法一：通过设置页面配置**

1. 点击左侧导航栏的 **"⚙️ 设置"**
2. 选择 **"🔑 API设置"** 标签页
3. 在"AI分析API"区域
4. AI提供商选择：硅基流动
5. 在"API密钥"输入框中粘贴您的API密钥
6. 点击 **"🧪 测试连接"** 按钮
7. 看到"连接成功"提示
8. 点击底部 **"💾 保存设置"** 按钮

**方法二：通过配置文件配置**

1. 找到项目文件夹中的 `config.yaml` 文件
2. 用记事本打开
3. 找到 `ai:` 部分
4. 修改为：
   ```yaml
   ai:
     api_key: "您的API密钥在这里"
   ```
5. 保存文件
6. 重启程序

### 6.3 AI分析结果解读

AI会返回以下内容：

#### 1️⃣ 基本面分析
- **趋势**：当前股票走势判断（上涨/下跌/震荡）
- **操作建议**：买入/持有/观望/卖出
- **风险等级**：低/中/高风险
- **支撑位**：价格下跌到这个位置可能止跌
- **阻力位**：价格上涨到这个位置可能遇阻

#### 2️⃣ 技术指标信号
- **均线趋势**：MA均线系统的信号
- **MACD信号**：MACD指标的买卖信号
- **RSI信号**：相对强弱指标信号
- **KDJ信号**：随机指标信号

#### 3️⃣ K线形态识别
- **形态**：识别出的K线形态（如：金叉、死叉、头肩底等）
- **置信度**：形态识别的可信度
- **交易信号**：该形态给出的买卖信号

#### 4️⃣ 决策仪表盘（高级功能）
- **精确点位**：买入价、止损价、目标价
- **检查清单**：多头排列、乖离安全、量能配合、趋势向上
- **风险提示**：特别需要注意的风险点

---

### 6.4 AI使用小技巧

1. **提问越具体，回答越准确**
   - 不好的问题："这只股票怎么样"
   - 好的问题："分析一下贵州茅台的技术面，看是否适合买入"

2. **结合多种分析**
   - 先用"技术分析"看走势
   - 再用"基本面分析"看价值
   - 最后用"决策仪表盘"看具体点位

3. **AI只是辅助，决策靠自己**
   - AI分析仅供参考
   - 请结合自己的判断做出投资决策
   - 投资有风险，入市需谨慎！

## 七、常见问题解答

### 7.1 安装和启动问题

#### Q1: 双击install.bat没反应？

**A:** 可能的原因和解决方法：
1. 确认您的电脑已安装Python 3.8或更高版本
2. 右键点击install.bat，选择"以管理员身份运行"
3. 检查是否有杀毒软件拦截

#### Q2: 程序启动后闪退？

**A:** 解决方法：
1. 查看项目文件夹下的 `logs` 目录，打开最新的日志文件
2. 尝试使用"轻量模式"启动
3. 删除 `config.yaml` 文件，重新启动
4. 确认已完整运行过 `install.bat`

#### Q3: 提示"Python未找到"？

**A:** 需要安装Python：
1. 访问 https://www.python.org/downloads/
2. 下载Python 3.10或更高版本
3. **重要**：安装时勾选 "Add Python to PATH"
4. 安装完成后重启电脑
5. 重新运行 `install.bat`

---

### 7.2 数据相关问题

#### Q4: 无法获取股票数据？

**A:** 检查以下几点：
1. 确认网络连接正常
2. 点击工具栏的"🔄 刷新"按钮
3. 程序会自动切换备用数据源，请稍等片刻
4. 如果仍然不行，运行 `data_source_wizard.py` 重新配置数据源

#### Q5: 数据刷新很慢？

**A:** 优化建议：
1. 使用"轻量模式"启动程序
2. 减少自选股数量（建议不超过10只）
3. 在设置中增加刷新间隔（如设置为60秒）
4. 关闭其他占用网络的程序

#### Q6: 搜索不到股票？

**A:** 检查：
1. 确认股票代码正确（6位数字）
2. 尝试用股票名称搜索
3. 确认该股票正在正常交易（未退市）
4. 检查网络连接

---

### 7.3 AI功能问题

#### Q7: AI功能不可用？

**A:** 检查：
1. 确认已配置API密钥（见第六章）
2. 检查API密钥是否正确
3. 确认网络可以访问API服务
4. 查看是否还有免费额度

#### Q8: AI分析返回错误？

**A:** 可能原因：
1. API密钥无效或已过期
2. 网络连接问题
3. API服务暂时不可用
4. 免费额度已用完

**解决方法：**
1. 检查API密钥是否正确
2. 检查网络连接
3. 稍后重试
4. 检查API服务商的免费额度

#### Q9: AI分析需要多长时间？

**A:** 正常情况下：
- 简单分析：5-10秒

#### Q10: 添加的自选股下次启动不见了？

#### Q12: 自选股最多可以添加多少只？
