## 第一步：安装Python

Python是运行这个程序的基础，就像汽车需要汽油一样。

### 1.1 下载Python

1. 打开浏览器（比如Chrome、Edge等）
2. 在地址栏输入：`https://www.python.org/downloads/`
3. 按回车键进入Python官网
4. 你会看到一个黄色的按钮，写着 **"Download Python"**（后面会跟着版本号，比如3.12.x）
5. 点击这个黄色按钮，开始下载

### 1.2 安装Python（最重要的一步！）

下载完成后，找到下载的文件（通常在"下载"文件夹里），双击打开。

**⚠️ 这一步非常关键，请仔细看！**

1. 安装窗口打开后，**首先看窗口最下面**
2. 找到一个选项：**"Add Python to PATH"**（把Python添加到环境变量）
3. **一定要勾选这个选项！！！**（不然后面会出问题）
4. 然后点击上面的 **"Install Now"**（现在安装）
5. 等待安装完成，可能需要几分钟
6. 看到 **"Setup was successful"**（安装成功）的提示后，点击 **"Close"** 关闭窗口

### 1.3 验证Python是否安装成功

1. 按下键盘上的 `Win + R` 键（Win键就是那个Windows徽标键，在Ctrl和Alt之间）
2. 在弹出的框里输入：`cmd`
3. 按回车键，会打开一个黑色的窗口（命令提示符）
4. 在黑色窗口里输入：`python --version`
5. 按回车键
6. 如果看到类似 `Python 3.12.0` 的文字，说明安装成功了！

## 第二步：下载项目

### 2.1 获取项目文件

如果你是从GitHub下载的：
1. 找到项目的GitHub页面
2. 点击绿色的 **"Code"** 按钮
3. 选择 **"Download ZIP"**（下载ZIP压缩包）
4. 等待下载完成

如果你已经有项目文件了，直接跳到下一步。

### 2.2 解压项目

1. 找到下载的ZIP文件（通常在"下载"文件夹里）
2. 右键点击这个文件
3. 选择 **"解压到当前文件夹"** 或者 **"解压到..."**
4. 等待解压完成
5. 解压后会得到一个文件夹，里面就是项目的所有文件

### 2.3 找个合适的位置放项目

建议把项目文件夹放在：
- 桌面（方便找到）
- D盘的某个文件夹（避免重装系统丢失）

**不要放在C盘的"Program Files"文件夹里，可能会有权限问题！**

## 第三步：安装依赖

依赖就是程序运行需要的一些工具，就像做饭需要油盐酱醋一样。

### 3.1 打开项目文件夹

1. 进入项目文件夹（就是刚才解压的那个文件夹）
2. 你会看到很多文件，找到一个叫 **`install.bat`** 的文件
3. 这个文件的图标通常是一个齿轮或者电脑的样子

### 3.2 运行安装脚本

1. **双击** `install.bat` 文件
2. 会弹出一个黑色的窗口，开始自动安装
3. 你会看到窗口里有文字在滚动，不用管它，等着就行
4. 安装过程可能需要5-10分钟，取决于你的网络速度
5. **不要关闭这个窗口！** 直到看到"安装完成"的提示

### 3.3 安装完成

当看到窗口里显示：
```
╔══════════════════════════════════════════════════╗
║           安装完成！                             ║
╠══════════════════════════════════════════════════╣
```

就说明安装成功了！按任意键关闭窗口即可。

## 第四步：启动程序

### 4.1 选择启动模式

项目文件夹里有几个启动脚本，根据你的电脑配置选择：

| 启动文件 | 适合电脑 | 说明 |
|---------|---------|------|
| **`一键启动-标准模式.bat`** | 内存≥4GB | 完整功能，推荐使用 |
| **`一键启动-轻量模式.bat`** | 内存<4GB | 简化功能，运行更快 |
| **`一键启动-Anaconda标准模式.bat`** | 使用Anaconda | 如果你装了Anaconda用这个 |
| **`一键启动-Anaconda轻量模式.bat`** | 使用Anaconda | Anaconda的轻量版 |

### 4.2 启动程序

1. 选择适合你的启动文件（新手推荐用 **`一键启动-标准模式.bat`**）
2. **双击** 这个文件
3. 会弹出一个黑色窗口，然后程序就启动了
4. 第一次启动可能需要几秒钟初始化，耐心等待

### 4.3 开始使用

程序启动后，你就可以：
- 查看股票行情
- 添加自选股
- 查看AI分析（需要配置API密钥，可选）

## 常见问题解答

### Q1: 双击install.bat没反应？

**可能原因：**
- 文件扩展名被隐藏了

**解决方法：**
1. 打开文件夹
2. 点击顶部的 **"查看"** 选项卡
3. 勾选 **"文件扩展名"**
4. 确认文件名确实是 `install.bat`，而不是 `install.bat.txt`

---

### Q2: 安装时提示"Python未找到"？

**可能原因：**
- Python没安装
- 安装时没勾选"Add Python to PATH"

**解决方法：**
1. 重新安装Python，**一定要勾选"Add Python to PATH"**
2. 安装完成后，重启电脑再试

---

### Q3: 安装依赖特别慢或者失败？

**可能原因：**
- 网络不好
- 使用的是国外的下载源

**解决方法：**
1. 检查网络连接是否正常
2. install.bat 会自动尝试使用国内镜像源，耐心等待
3. 如果还是失败，可以试试手机开热点给电脑用

---

### Q4: 启动程序后闪退？

**可能原因：**
- 依赖没安装好
- 配置文件有问题

**解决方法：**
1. 先确认 install.bat 运行成功了
2. 试试用轻量模式启动
3. 还是不行的话，删除项目文件夹里的 `config.yaml` 文件，再重新启动

---

### Q5: 程序能打开，但看不到股票数据？

**可能原因：**
- 网络连接问题
- 数据源服务器暂时不可用

**解决方法：**
1. 检查网络是否正常
2. 等一会儿再试
3. 程序会自动切换备用数据源，不用着急

---

### Q6: 如何配置AI功能？（可选）

AI功能不是必须的，没有它也能看股票行情。如果你想用：

1. 访问：`https://siliconflow.cn/`
2. 注册一个账号（免费）
3. 登录后，在控制台找到API密钥
4. 复制这个密钥
5. 打开项目文件夹里的 `config.yaml` 文件（用记事本打开）
6. 找到 `api_key: ""` 这一行
7. 把你的API密钥填在引号里，比如：`api_key: "你的密钥"`
8. 保存文件，重启程序
